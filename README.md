# FocusWind Distribution

Canal publico de distribuicao dos releases oficiais do FocusWind.

## Website oficial

- [www.focuswind.online](https://www.focuswind.online)

## Empresa responsavel

- [Borini Software](https://www.borinisoftware.com.br)

## O que este repositorio publica

- instaladores publicos do FocusWind
- arquivos de checksum (`SHA-256`)
- notas curtas de distribuicao quando necessario

## Fonte oficial de produto

O site oficial do FocusWind e:

- [www.focuswind.online](https://www.focuswind.online)

Todos os links de download publicados pelo site devem apontar para assets deste repositorio ou para canais oficiais explicitamente informados pela equipe.

## Integridade

Sempre que houver release publico, publique tambem o arquivo de checksum correspondente para validacao manual de integridade.
